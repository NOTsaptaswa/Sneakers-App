This is a mobile e-commerce application built using Flutter with a focus on user experience, clean architecture, and smooth custom animations. It includes product browsing, category view, cart management, and real-time state updates.

## Overview
The main objective of this project was to practice UI design, state management, and performance optimization using Flutter. The app showcases smooth animations, reusable components, and structured state handling using Provider.

## Key Features
- Product listing with categorization
- Real-time cart and pricing updates
- Custom UI animations (Hero transitions, TweenSequence)
- High performance with optimized widget usage
- Centralized theming and reusable design components
- Domain-driven data models

## Technologies Used
- Flutter, Dart
- Provider (state management)
- Custom Animations: Hero, TweenSequence
- Android Studio / VS Code
- Domain-driven design approach

## What I Learned
- Managing complex state using Provider
- Building reusable widget components
- Designing smooth, interactive animations
